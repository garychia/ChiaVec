ChiaVec
ChiaVec is a powerful tool that offers sequence containers, namely Vec and CudaVec. Vec allocates memory on CPUs while CudaVec allocates memory on GPUs using CUDA. ChiaVec performs memory management across different processing units under the hood. This makes it easy for high-performance computing tasks using C++. C and Rust bindings are also provided.

Detailed Description

template<class T, class Allocator = Memory::DefaultAllocator>
class ChiaVec::Raw::RawVec< T, Allocator >

A simple vector-like class that manages raw memory.

The RawVec class is a simple vector-like class that manages raw memory using a custom memory allocator. It provides basic functionality for allocating, resizing, and copying memory, as well as some convenience methods for working with the raw memory.

Template Parameters
TThe type of elements stored in the vector.
AllocatorThe memory allocator to use for managing the raw memory. Defaults to Memory::DefaultAllocator.
