class Chia\+Vec\+::\+Raw\+::\+Raw\+Vec$<$ T, Allocator $>$}

A simple vector-\/like class that manages raw memory. 

The {\ttfamily \mbox{\hyperlink{classChiaVec_1_1Raw_1_1RawVec}{Raw\+Vec}}} class is a simple vector-\/like class that manages raw memory using a custom memory allocator. It provides basic functionality for allocating, resizing, and copying memory, as well as some convenience methods for working with the raw memory.
