Adds a new element to the end of the vector. 

This function appends the given {\ttfamily element} to the end of the vector. If the vector\textquotesingle{}s capacity needs to be increased to accommodate the new element, the function will automatically resize the underlying storage.

The {\ttfamily on\+Host} parameter specifies whether the {\ttfamily element} is located on the host (CPU) or the device (GPU). If {\ttfamily on\+Host} is true and the allocator is set to allocate on the host, the element is directly assigned to the vector\textquotesingle{}s storage. Otherwise, the element is copied using the allocator\textquotesingle{}s {\ttfamily copy()} function.
